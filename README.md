# Install instructions

You'll need a python interpreter and use the commands :

```
pip install pygame
```
than
```
pip install pytmx
```
than
```
pip install pyscroll
```

and pass the `main.py` file in the interpretor by using :

```
python main.py
```
you have a window 



you can play by using those keyboard keys
